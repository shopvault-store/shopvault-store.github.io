Pokazivači (za one koji su već upoznati sa C-om)

Hajde da počnemo od jednog vrlo lakog primera sa pokazivačima:


#include <stdio.h>

int main()
{
    int x = 5;
    int *p = &x;
    int **pp = &p; // pp pokazuje na p
    **pp = 20;     // menja x preko pp
    
    printf("x: %d\n", x);
    printf("p: %d\n", *p);
    printf("pp: %d\n", **pp);
    
    return 0;
}

/*
Ispis na konzolu:
x: 20
p: 20
pp: 20
*/

E sad, zašto je baš ovakav ispis? Pa zato što je pp pokazivac na pokazivac na x. Ali dobro, gde to piše? Pa drugom linijom smo definisali da p pokazuje na x (int *p = &x;) a pritom pp pokazuje na p (int **pp = &p;). Samim tim, pp --> p --> x. Ali čekajte, nije to sve sa pokazivačima, tek smo zagrebali po površini.


Sledeći primer je neispravan, jer samo dupli pokazivač može da pokazuje na pokazivač, odnosno mora biti deklarisan kao pokazivač na pokazivač 


#include <stdio.h>

int main()
{
    int x = 5;
    int *p = &x;
    int *pp = &p; // pp pokazuje na p
    *pp = 20;     // menja x preko pp
    
    printf("x: %d\n", x);
    printf("p: %d\n", *p);
    printf("pp: %d\n", *pp);
    
    return 0;
}
